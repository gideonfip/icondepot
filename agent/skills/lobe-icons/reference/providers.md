## Variant Types

| Variant Suffix | Meaning | Example |
|--------------|---------|---------|
| (none/mono) | Monochrome | `openai.svg` |
| `-color` | Full color | `openai-color.svg` |
| `-text` | Logo + text (English) | `openai-text.svg` |
| `-text-cn` | Logo + text (Chinese) | `zhipu-text-cn.svg` |
| `-brand` | Brand style | `openai-brand.svg` |
| `-brand-color` | Brand + color | `openai-brand-color.svg` |

## Quick Reference

```tsx
import { toc } from '@lobehub/icons';

// Find all provider icons
const providers = toc.filter((i) => i.group === 'provider');

// Find all icons that have color variants
const colorIcons = toc.filter((i) => i.param.hasColor);

// Find icon by id
const gemini = toc.find((i) => i.id === 'Gemini');
```
